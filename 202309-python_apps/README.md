## Documented Installation Commands

```bash
# black - https://github.com/psf/black#installation
pip install black

# codespell - https://github.com/codespell-project/codespell#installation
pip install codespell

# copier - https://github.com/copier-org/copier#installation
pipx install copier

# cruft - https://github.com/cruft/cruft#installation
pip3 install cruft

# csvkit - https://csvkit.readthedocs.io/en/latest/tutorial/1_getting_started.html#installing-csvkit
sudo pip install csvkit

# databricks-cli - https://github.com/databricks/databricks-cli#installation
pip install --upgrade databricks-cli

# doit - https://github.com/pydoit/doit#install
pip install doit

# flake8 - https://flake8.pycqa.org/en/latest/index.html#installation
python -m pip install flake8

```
